# Бизнес требования

## 1. Введение
**Приложение:** Система выставления оценок студентам
**Цель:** Автоматизация процесса выставления и учета оценок в учебных заведениях

## 2. Бизнес-цели
- Упрощение процесса выставления оценок преподавателями
- Централизованное хранение данных об успеваемости
- Предоставление студентам доступа к своим оценкам
- Автоматизация формирования отчетов

## 3. Основные требования
### 3.1 Функциональные
- Регистрация пользователей (преподаватели, студенты, администраторы)
- Создание курсов и групп студентов
- Выставление оценок по различным критериям
- Просмотр статистики успеваемости
- Формирование отчетов (ведомости, рейтинги)

### 3.2 Нефункциональные
- **Производительность:** Поддержка до 1000 пользователей одновременно
- **Безопасность:** Авторизация по ролям, шифрование данных
- **Надежность:** Доступность 99.5%
- **Удобство использования:** Интуитивный интерфейс

## 4. Целевая аудитория
- Преподаватели вузов и колледжей
- Студенты
- Администраторы учебных заведений

## 5. Ограничения
- Поддержка только веб-интерфейса
- Интеграция только с существующими СУБД
- Мультиязычность: русский и английский языки
